# AI Career Mentor Constitution

## Purpose
AI Career Mentor helps users discover suitable career paths through AI-powered recommendations and skill-gap analysis.

## Principles

### User First
All recommendations should prioritize user benefit and career growth.

### Transparency
AI-generated recommendations should be explainable and understandable.

### Privacy
User information must be handled securely and responsibly.

### Maintainability
Code should be modular, documented, and easy to maintain.

### Quality
All features should be tested before deployment.

## Development Standards

- Use clear naming conventions.
- Follow project coding standards.
- Maintain documentation for major features.
- Review changes before merging.
